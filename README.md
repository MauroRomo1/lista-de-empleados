## Para colonar este proyecto copie y pege uno por uno los siguientes comandos en una termina
- Paso 1:
```
git clone https://github.com/MauroRomo1/lista-de-empleados.git
```

- Paso 2:
```
npm install
```
- Paso 3:
```
npm run dev
```
